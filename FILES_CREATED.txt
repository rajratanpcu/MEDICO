================== API REQUIREMENTS ====================

ENDPOINT 1: POST /api/reports/upload
- Accepts: FormData with 'file' and 'patientId'
- Returns: { fileId, fileName, fileSize, uploadedAt, patientId }

ENDPOINT 2: POST /api/ai/analyze-report
- Accepts: { fileId, patientId?, reportType? }
- Returns: { summary, confidence, abnormal_values, findings, recommendations }
